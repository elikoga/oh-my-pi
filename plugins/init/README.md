# @oh-my-pi/init

Initialize AGENTS.md documentation for a codebase.

## Commands

### `/init`

Analyzes the codebase and creates an AGENTS.md file with:

- Build, lint, and test commands
- High-level code architecture
- Important context from existing documentation (README.md, Cursor rules, Copilot rules)

The generated file provides guidance to future Claude Code instances operating in the repository.
